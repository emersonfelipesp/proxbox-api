# proxbox_api/generated/proxmox Directory Guide

## Purpose

Stores Proxmox API Viewer derived artifacts generated by `proxbox_api.proxmox_codegen`.

## Typical Files

- `raw_capture.json`: Raw endpoint/method capture from Playwright and parser merge.
- `openapi.json`: Generated OpenAPI 3.1 schema.
- `pydantic_models.py`: Generated Pydantic v2 model source.

## Extension Guidance

- Re-generate artifacts instead of editing them by hand.
