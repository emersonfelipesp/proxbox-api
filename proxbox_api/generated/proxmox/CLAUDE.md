# proxbox_api/generated/proxmox Directory Guide

## Purpose

Stores Proxmox API Viewer derived artifacts generated by `proxbox_api.proxmox_codegen`.

## Typical Files

- `<version-tag>/raw_capture.json`: Raw endpoint/method capture from Playwright and parser merge.
- `<version-tag>/openapi.json`: Generated OpenAPI 3.1 schema.
- `<version-tag>/pydantic_models.py`: Generated Pydantic v2 model source.
- `<version-tag>/crawl_checkpoint.json`: Async crawl checkpoint artifact.

## Version Tag Policy

- `latest` must only be generated from official `https://pve.proxmox.com/pve-docs/api-viewer/`.
- Version-specific local clusters must use explicit tags such as `8.3.0`.

## Extension Guidance

- Re-generate artifacts instead of editing them by hand.
