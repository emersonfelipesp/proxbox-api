# proxbox_api/generated/proxmox Directory Guide

## Purpose

Stores Proxmox API Viewer derived artifacts generated by `proxbox_api.proxmox_codegen`.

## Typical Files

- `<version-tag>/raw_capture.json`: raw endpoint and method capture from Playwright and parser merge.
- `<version-tag>/openapi.json`: generated OpenAPI 3.1 schema.
- `<version-tag>/pydantic_models.py`: generated Pydantic v2 model source.
- `<version-tag>/crawl_checkpoint.json`: crawl checkpoint artifact for resuming generation.
- `latest/`: canonical generated snapshot for the current official API viewer.
- `runtime_generated_routes_cache.json`: cache file written by runtime route generation.

## Version Policy

- `latest` should represent the official Proxmox API viewer snapshot.
- Version-specific snapshots such as `8.3.0/` should match the exact upstream version they were generated from.
- Do not mix manual edits with generated content inside this tree.

## Extension Guidance

- Regenerate artifacts instead of editing them by hand.
- Keep `latest` and explicit versioned snapshots aligned with generator output.
- If you need to change generation behavior, update the generator code in `proxbox_api/proxmox_codegen/` first.
