# tasks/ Directory Guide

## Purpose

Development task tracking for the `proxbox-api` project. This directory is not part of the application runtime.

## Files

| File | Role |
|------|------|
| `todo.md` | Active task list with checkable items, progress notes, and review sections |

## Conventions

- Add planned work as checklist items: `- [ ] <task description>`
- Mark completed items: `- [x] <task description>`
- Add a `## Review` section at the bottom of completed task groups with lessons learned.
- This file is for developer coordination — do not import or reference it from application code.
